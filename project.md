# Калькулятор для риелторов BARAKAT

## Описание проекта
Одностраничный веб-калькулятор (single-page HTML-приложение) для расчёта рассрочки на недвижимость по принципу мурабаха (без процентов, с фиксированной наценкой), как того требует шариатский стандарт. Инструмент предназначен для риелторов, помогающих клиентам подобрать и рассчитать условия покупки квартиры в рассрочку, а также сформировать готовый текст с расчётом для отправки клиенту в WhatsApp/Telegram.

## Для кого
- Только для риелторов, работающих с исламской рассрочкой (Чечня). Клиентам доступ к самому инструменту не предоставляется — риелтор использует его сам, а готовый расчёт отправляет клиенту в виде текста (через WhatsApp/Telegram).

## Какие страницы (вкладки) уже есть
Проект реализован как одна HTML-страница с переключаемыми вкладками:
1. **Калькулятор** — ввод параметров объекта (площадь, цена за кв.м, итого стоимость, этаж, вид из окна, локация), выбор срока рассрочки (1–6 лет), наценки и первоначального взноса; расчёт ежемесячного платежа и итоговой суммы с учётом досрочного погашения.
2. **График** — детальный график платежей по месяцам с остатком долга, сводка по объекту и контактам, кнопка копирования расчёта для отправки клиенту.
3. **Сроки** — сравнительная таблица ежемесячных платежей и переплаты для всех сроков от 1 до 6 лет.
4. **Подбор** — добавление и сравнение до 4 объектов недвижимости, выделение самого выгодного варианта по размеру ежемесячного платежа.
5. **Контакты** — данные риелтора (ФИО, WhatsApp, Telegram) и клиента.

## Какие страницы планируются
- **База клиентов** — сохранение и управление списком клиентов риелтора.
- **Экспорт в PDF** — формирование расчёта/графика платежей в виде PDF-документа.
- **Договор** — генерация договора (рассрочки/мурабаха) на основе данных расчёта.

## Стиль дизайна
- Цветовая палитра: тёмно-зелёный (#1a472a, #0d2b1a) + золотой акцент (#d4af37) — отсылка к исламской/премиальной эстетике.
- Светлый фон (#f4f4f0), карточки с белым фоном, скруглённые углы (8–12px), минималистичные тени отсутствуют — плоский дизайн.
- Заголовок (Playfair Display) для имени риелтора — премиальный, "визитка"-стиль.
- Адаптивная сетка (grid), мобильная оптимизация (медиа-запрос для экранов до 480px).
- Компактный, плотный UI с маленькими шрифтами (10–14px), характерный для мобильных рабочих инструментов.

## Технический стек
- Чистый HTML/CSS/JS, без фреймворков и зависимостей (кроме Google Fonts).
- Один файл `al-barakat (1).html`.
