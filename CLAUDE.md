# CLAUDE.md

Правила работы с проектом «Калькулятор для риелторов BARAKAT».

## Структура
- Всё делаем в одном HTML-файле (`al-barakat (1).html`) — HTML, CSS и JS вместе, без отдельных файлов.
- Никаких внешних библиотек и фреймворков (кроме уже подключённого Google Fonts). Только чистый HTML/CSS/JS.

## Дизайн
- Сохраняем текущий стиль: тёмно-зелёный + золотой, светлый фон, плоские карточки со скруглёнными углами.
- Не менять цветовую палитру, шрифты и общий вид без явной просьбы.

## Вёрстка
- Адаптивность обязательна — всё должно нормально выглядеть и работать на мобильных (как и сейчас, через media-запросы).

## Текст
- Простой, дружеский тон, без канцелярита и формальных оборотов.

## Рабочий процесс
- Перед созданием нового файла — сначала рассказать, что планируется сделать, и дождаться подтверждения.
